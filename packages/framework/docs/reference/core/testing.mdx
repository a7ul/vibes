All testing utilities are exported from `@vibes/framework`. Tests run in milliseconds and never hit real APIs.

## `TestModel`

A schema-aware mock model. It understands your agent's tools and output schema, so you don't have to manually construct raw SDK response objects.

```ts
import { TestModel } from "@vibes/framework";
```

### Options

| Option      | Type      | Default | Description                                                   |
| ----------- | --------- | ------- | ------------------------------------------------------------- |
| `text`      | `string`  | -       | Return this text as the response                              |
| `callTools` | `boolean` | `true`  | If true, calls all tools on turn 1, then produces final output |

### Auto-behavior (default)

With no options, `TestModel` auto-generates a valid agent run:
- **Turn 1**: Calls every non-`final_result` tool once with schema-valid arguments
- **Turn 2**: Calls `final_result` with schema-valid data, or returns text if no `outputSchema`

```ts
// Smoke test - just verify tools run without errors
Deno.test("all tools execute", async () => {
  const result = await agent
    .override({ model: new TestModel() })
    .run("test");
  assertExists(result.output);
});
```

### Specific text response

```ts
const model = new TestModel({ text: "hello world" });
const result = await agent.override({ model }).run("say hello");
assertEquals(result.output, "hello world");
```

### Skip tool calls

```ts
const model = new TestModel({ callTools: false });
// Goes straight to text/structured output without calling tools
```

## `FunctionModel`

Full programmatic control - supply a function that receives the current prompt state and returns a response.

```ts
import { FunctionModel } from "@vibes/framework";
```

The function receives `{ messages, tools, turn }` and returns a response object:

```ts
const model = new FunctionModel(({ messages, tools, turn }) => {
  const lastUser = messages.findLast((m) => m.role === "user");
  const text = typeof lastUser?.content === "string" ? lastUser.content : "";

  return {
    text: text.includes("hello") ? "Hi there!" : "I don't understand.",
    finishReason: "stop" as const,
    usage: { inputTokens: 5, outputTokens: 3 },
  };
});
```

Simulate a tool call on turn 1, then text on turn 2:

```ts
const model = new FunctionModel(({ turn }) => {
  if (turn === 0) {
    return {
      toolCalls: [{
        toolCallType: "function" as const,
        toolCallId: "tc1",
        toolName: "get_weather",
        args: JSON.stringify({ city: "Tokyo" }),
      }],
      finishReason: "tool-calls" as const,
      usage: { inputTokens: 10, outputTokens: 5 },
    };
  }
  return {
    text: "The weather in Tokyo is 22°C.",
    finishReason: "stop" as const,
    usage: { inputTokens: 20, outputTokens: 10 },
  };
});
```

## `MockLanguageModelV3` (from `ai/test`)

For detailed response control, use `MockLanguageModelV3` directly from the AI SDK:

```ts
import { MockLanguageModelV3, mockValues, convertArrayToReadableStream } from "ai/test";
```

### Plain text response

```ts
Deno.test("basic text run", async () => {
  const model = new MockLanguageModelV3({
    doGenerate: () => Promise.resolve({
      text: "hello world",
      finishReason: "stop",
      usage: { inputTokens: 10, outputTokens: 5 },
      rawCall: { rawPrompt: null, rawSettings: {} },
    }),
  });

  const agent = new Agent({ model });
  const result = await agent.run("Say hello.");
  assertEquals(result.output, "hello world");
  assertEquals(result.usage.requests, 1);
});
```

### Multi-turn with `mockValues`

`mockValues` cycles through values one at a time - use it for multi-turn scenarios:

```ts
Deno.test("tool call then text", async () => {
  type R = Awaited<ReturnType<MockLanguageModelV3["doGenerate"]>>;

  const doGenerate = mockValues<R>(
    // Turn 1: model calls the tool
    {
      toolCalls: [{
        toolCallType: "function",
        toolCallId: "tc1",
        toolName: "get_weather",
        args: JSON.stringify({ city: "Tokyo" }),
      }],
      finishReason: "tool-calls",
      usage: { inputTokens: 10, outputTokens: 5 },
      rawCall: { rawPrompt: null, rawSettings: {} },
    },
    // Turn 2: text after seeing tool result
    {
      text: "The weather in Tokyo is 22°C and sunny.",
      finishReason: "stop",
      usage: { inputTokens: 20, outputTokens: 10 },
      rawCall: { rawPrompt: null, rawSettings: {} },
    },
  );

  const model = new MockLanguageModelV3({
    doGenerate: () => Promise.resolve(doGenerate()),
  });

  const agent = new Agent({ model, tools: [weatherTool] });
  const result = await agent.run("Weather in Tokyo?");
  assertEquals(result.usage.requests, 2);
});
```

### Streaming

```ts
Deno.test("stream text", async () => {
  const model = new MockLanguageModelV3({
    doStream: () => Promise.resolve({
      stream: convertArrayToReadableStream([
        { type: "text-delta", textDelta: "hello " },
        { type: "text-delta", textDelta: "world" },
        { type: "finish", finishReason: "stop", usage: { inputTokens: 10, outputTokens: 5 } },
      ]),
      rawCall: { rawPrompt: null, rawSettings: {} },
    }),
  });

  const agent = new Agent({ model });
  const stream = agent.stream("Say hello.");

  let collected = "";
  for await (const chunk of stream.textStream) {
    collected += chunk;
  }

  assertEquals(collected, "hello world");
});
```

## `setAllowModelRequests()`

Call this at the top of a test file to block any non-mocked agent run from making real API calls:

```ts
import { setAllowModelRequests } from "@vibes/framework";

setAllowModelRequests(false); // throws ModelRequestsDisabledError for real calls

// Use agent.override({ model: mockModel }) for the runs you do want to execute
```

## `captureRunMessages()`

Wrap any agent call to record the exact messages sent to the model on each turn:

```ts
import { captureRunMessages } from "@vibes/framework";

const { result, messages } = await captureRunMessages(() =>
  agent.run("Hello", { deps: myDeps })
);

// messages[0] = messages array for turn 1
// messages[1] = messages array for turn 2
const systemMsg = messages[0].find((m) => m.role === "system");
assertEquals(systemMsg?.content?.includes("Alice"), true);
```

Useful for verifying dynamic system prompts, history processors, and message structure.
